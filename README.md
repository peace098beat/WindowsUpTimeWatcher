
# WindowsUpTimeWatcher
Windowsの起動時間を簡易計測するアプリケーション

## Description
Windowsの起動時間を簡易計測する. 例えば、HDDからSSDに換装した際などの計測に利用.
アプリ起動時に勝手にスタートアップに登録され、
Windowsの次回起動時にアプリが自動起動され、そのときの経過時間を表示する「だけ」

## 機能
 - [x] 起動時間の表示
 - [x] スタートアップに登録
 - [ ] 起動時間の記録
 - [ ] スタートアップに登録を任意に
